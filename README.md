# EigenFit Frontend

Plataforma moderna y minimalista para la gestión de gimnasios y centros deportivos.

## Tech Stack

- **React** (TypeScript)
- **Vite** (Build Tool)
- **Tailwind CSS** (Styling)
- **Lucide React** (Icons)
- **React Router DOM** (Routing)

## Cómo iniciar

1.  **Instalar dependencias:**
    ```bash
    bun install
    # o
    npm install
    ```

2.  **Correr servidor de desarrollo:**
    ```bash
    bun run dev
    # o
    npm run dev
    ```

3.  Abrir `http://localhost:5173` en tu navegador.

## Estructura del Proyecto

- `/components`: Componentes reutilizables y vistas del dashboard.
- `/dashboard`: Layout principal y vistas específicas (Inicio, Visitas, Tablas, Configuración).
- `/ui`: Primitivas de UI (Botones, Modales, Inputs, etc).
